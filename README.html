SUMERU is a RISC-V (rv32im) single board computer.

This SDK is based on a specific version of the stock RISC-V GCC 9.2.0 toolchain and the associated (newlib) libc. Both of these are published by the RISC-V foundation.

For more information refer to:

www.riscv.org
github.com/riscv/riscv-gcc
www.purpe.com/sumeru

This SDK includes a binary toolchain for Linux AMD-64 (Debian 10). Instructions for compiling the toolchain locally are in doc/toolchain.html
