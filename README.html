SUMERU is a RISC-V (rv32im) single board computer.

This SDK is based on the stock RISC-V toolchain and the libc (newlib), both, provided by the RISC-V foundation.

For more information refer to:

www.riscv.org
github.com/riscv/riscv-gcc
www.purpe.com/sumeru
www.purpe.com/sumeru-sdk

Instructions for compiling the toolchain locally are available at doc/toolchain.html

SDK documentation is at:

QUICKSTART.html
doc/index.html
doc/sdk-tools-index.html
doc/sdk-environment.html

Mailing list: sumeru-sdk (a) purpe.com
